# 📊 Global Layoffs Exploratory Data Analysis using SQL

## 📌 Overview
This project explores a global layoffs dataset using SQL to uncover trends and business insights across companies, industries, countries, funding stages, and time.

## 🛠️ Tools
- SQL
- MySQL
- Git
- GitHub

## 📚 SQL Concepts Used
- Aggregate Functions
- GROUP BY
- ORDER BY
- CTEs
- Window Functions
- DENSE_RANK()
- Rolling Totals

## 📈 Analysis Performed
- Data Exploration
- Company-wise Analysis
- Industry-wise Analysis
- Country-wise Analysis
- Year-wise Analysis
- Funding Stage Analysis
- Monthly Rolling Total
- Top Companies by Layoffs
